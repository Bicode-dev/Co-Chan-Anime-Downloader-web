Comment Ça Marche

Étape 1: Installation

Sur Windows, exécutez Setup.bat. Sur Android avec Termux, utilisez bash Install-termux-shorcut-set.sh.

Étape 2: Lancement

Ouvrez le script Anime-dowload.py ou utilisez la ligne de commande.

Étape 3: Téléchargement

Exécutez une commande comme: python Anime-dowload.py "one piece" vf

python Anime-dowload.py "nom de l'anime" version

Étape 4: Profitez

Vos animes sont téléchargés et prêts à être visionnés !

Étape 5: Support

Pour toute question, rejoignez le serveur Discord.

Questions Fréquentes (FAQ)

Qu'est-ce que Co-Chan ?
Co-Chan est un téléchargeur d'anime qui extrait directement les animes depuis des serveurs russes via anime-sama.
Comment installer sur Windows ?
Exécutez Setup.bat pour une installation facile.
Est-ce compatible avec Android ?
Oui, via Termux en exécutant bash Install-termux-shorcut-set.sh.
Quelles versions sont supportées ?
VF et VOSTFR, en fonction de la disponibilité sur anime-sama.
Où trouver de l'aide ?
Utilisez --help ou rejoignez notre Discord.
Le projet est-il open-source ?
Oui, disponible sur GitHub.
Y a-t-il des limitations ?
Fonctionne uniquement si le lecteur sibnet est disponible.
Comment contribuer ?
Forkez le repo sur GitHub et soumettez une pull request.
Est-ce gratuit ?
Oui, entièrement gratuit et open-source.
Supporte-t-il d'autres langues ?
Actuellement en français uniquement.
