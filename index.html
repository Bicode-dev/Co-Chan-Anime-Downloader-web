Bienvenue sur Co-Chan Anime Downloader

Co-Chan est un téléchargeur d'anime qui extrait directement les animes depuis des serveurs russes. Il se base sur l'indexation de anime-sama et fonctionne uniquement si le lecteur sibnet est disponible sur le site. Le projet est entièrement en français.

Fonctionnalités Principales

  • Téléchargement d'animes en VF (Version Française) ou VOSTFR (Version Originale Sous-Titrée en Français).
  • Extraction directe depuis des serveurs russes via anime-sama.
  • Utilisation en ligne de commande pour spécifier le nom de l'anime et la version (ex: python Anime-dowload.py "one piece" vf).
  • Affichage d'aide avec l'option --help.

Comment Ça Marche

Le script principal est Anime-dowload.py. Vous pouvez l'utiliser via la ligne de commande ou l'ouvrir directement. Pour toute question, rejoignez le serveur Discord.

Technologies Utilisées

  • Python pour le script principal.
  • Bash pour l'installation sur Termux (Android).
  • Batch pour l'installation sur Windows (Setup.bat).

Instructions d'Installation

Sur Termux : Exécutez bash Install-termux-shorcut-set.sh.
Sur Windows : Ouvrez Setup.bat ou utilisez le script en ligne de commande.
